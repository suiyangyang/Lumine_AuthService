# Lumine AuthServer 管理后台基础接口说明

> 状态：Current
> 更新时间：2026-06-09
> 适用范围：后台管理 API、登录注册接口和 Swagger 调试

## 初始化数据

- 启动时会自动执行数据库迁移，并初始化管理员、管理员角色、基础权限和默认 OIDC 客户端。
- 默认种子配置位于 `Lumine.AuthServer/appsettings.json` 的 `SeedData` 节点。
- 默认管理员账号为 `admin`，密码从 `SeedData:AdminPassword` 或环境变量 `SeedData__AdminPassword` 读取。

## Swagger 调试

- 开发环境启动后访问 Swagger UI。
- 先调用 `POST /api/auth/login` 获取 JWT。
- 点击右上角 `Authorize`，输入 `Bearer {token}` 后即可调试受保护接口。
- 每个受保护接口的 Swagger 说明中会显示“所需权限”，并自动标注 `401` / `403` 响应含义。

## 核心接口

### 认证

- `POST /api/auth/login`：登录并返回 `access_token`、`id_token`、角色与权限摘要。
- `POST /api/auth/register`：注册用户。

### 仪表盘

- `GET /api/dashboard/summary`：返回用户、角色、权限、客户端数量，以及最近授权与 token 趋势摘要。

### 用户

- `GET /api/users`：分页查询用户。
- `GET /api/users/{id}`：查询用户详情。
- `GET /api/users/me`：查询当前登录用户。
- `PUT /api/users/me`：修改当前登录用户资料。
- `POST /api/users/me/change-password`：修改当前登录用户密码。
- `POST /api/users`：创建用户。
- `PUT /api/users/{id}`：修改用户资料。
- `PUT /api/users/{id}/status`：启用或停用用户。
- `POST /api/users/{id}/reset-password`：重置用户密码。
- `DELETE /api/users/{id}`：删除用户。
- `PUT /api/users/{id}/roles`：覆盖用户角色集合。
- `POST /api/users/{id}/roles/{roleId}`：给用户追加单个角色。
- `DELETE /api/users/{id}/roles/{roleId}`：移除用户单个角色。

### 角色

- `GET /api/roles`
- `GET /api/roles/{id}`
- `POST /api/roles`
- `PUT /api/roles/{id}`
- `DELETE /api/roles/{id}`
- `PUT /api/roles/{id}/permissions`
- `POST /api/roles/{id}/permissions/{permissionId}`
- `DELETE /api/roles/{id}/permissions/{permissionId}`

### 权限

- `GET /api/permissions`
- `GET /api/permissions/{id}`
- `POST /api/permissions`
- `PUT /api/permissions/{id}`
- `DELETE /api/permissions/{id}`

### OIDC 客户端

- `GET /api/clients`
- `GET /api/clients/{id}`
- `POST /api/clients`
- `PUT /api/clients/{id}`
- `DELETE /api/clients/{id}`

## 权限要求

| 模块 | 查看权限 | 管理权限 |
| --- | --- | --- |
| 用户 | `users.view` | `users.manage` |
| 角色 | `roles.view` | `roles.manage` |
| 权限 | `permissions.view` | `permissions.manage` |
| OIDC 客户端 | `clients.view` | `clients.manage` |

说明：

- `/api/users/me`、`/api/users/me/change-password` 只要求用户已登录。
- `/api/dashboard/summary` 依赖当前 token 中的权限集合裁剪返回内容。

## 统一响应约定

- 未登录访问受保护接口时，返回统一 `401` JSON：包含 `statusCode`、`error`、`message`、`path`、`timestamp`。
- 已登录但权限不足时，返回统一 `403` JSON，并附带 `requiredPermission` 字段。
- 登录支持历史明文密码兼容；旧账号首次成功登录后会自动升级为哈希密码。
- 如需重置默认管理员密码，可将 `SeedData:ForceUpdateAdminPassword` 临时设为 `true` 后重启服务。
